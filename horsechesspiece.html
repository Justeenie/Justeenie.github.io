Horse Chess Piece

3D printing and it's applications

3D printing is a relatively recent development in technology where an object can be modeled on a computer software and the digital file could then be translated into printing instructions in order to construct the model. The process of 3D printing can be simplified to 3 steps.

  1. Firstly, a 3D model must be designed using a modeling software such as Solidworks, TinkerCAD, Autodesk Inventor. In this module, I was taught how to use Fusion360 to design a 3d model.
  2. The second step is to prepare the model for printing such as adding supports to the model and adjusting the printer's settings.
  3. The last step is to simply save the file into a memory device such as an SD card or a USB thumb drive in order to transfer the file to the printer.

For this module, we were given the assignment to create a 3D model of a Horse Chess Piece on Fusion 360, prepare the model for printing using CURA, and then print it out using the Ultimaker 2 printer in the FabLab.
